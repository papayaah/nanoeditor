# Product Overview

Nano Editor is a dual-purpose application with two main features:

1. **Document Editor** - AI-assisted rich text document editing
2. **Social Post Creator** - AI-powered social media post generation and refinement

Both features use Chrome's built-in AI APIs for intelligent writing assistance.

## Core Features

### Document Editor
- Rich text editing with BlockNote editor
- AI writing assistance (rewrite, continue writing)
- PDF export functionality
- Markdown import/export
- Local-first storage using IndexedDB

### Social Post Creator
- Generate social media posts with AI
- Refine and iterate on post variations
- Customize tone, style, and platform
- Track post history and submissions
- Local storage of all posts

### Shared Features
- Dark mode support
- Completely client-side (no backend)
- Local-first data storage

## Key User Flows

1. **Document editing**: Create and edit rich text documents with AI assistance for rewriting and content generation
2. **Social post creation**: Generate social media posts using AI with customizable settings and multiple variations
3. **Export**: Export documents as PDF or markdown

## Target Environment

- Chrome browser with built-in AI APIs enabled
- Client-side only (no backend)
- Deployed on Vercel
