# Social Media Post Helper

A two-column AI-powered tool to help craft social media posts.

## Features

- **Left Column**: Write your original sentence
- **Right Column**: Get 3 AI-generated variations automatically
- **Customizable Settings**: Choose tone (casual/neutral/formal) and length (short/medium/long)
- **One-Click Copy**: Copy any suggestion to clipboard

## How It Works

1. Type your sentence in the left column
2. End with punctuation (. ! ?)
3. AI generates 3 variations instantly
4. Click copy button to use any suggestion

## Requirements

- Chrome Canary/Dev with built-in AI (Writer API)
- Chrome AI features must be enabled

## Usage

Access via the "Post Helper" button in the sidebar or navigate to `/posts`
