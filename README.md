# Nano Editor

A minimal document editor built with [BlockNote](https://www.blocknotejs.org/) and powered by Chrome's built-in AI.

## Run Locally

```bash
npm install
npm run dev
```

## Component Documentation

View the component library in Storybook:

```bash
npm run storybook
```

This showcases the `BlockNoteEditor` and `PostCreator` components with interactive examples, perfect for understanding how to use them in your own projects. See [STORYBOOK.md](STORYBOOK.md) for more details.
